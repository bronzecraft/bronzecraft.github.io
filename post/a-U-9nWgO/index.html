Tkinter库学习笔记（1）

2021-01-31 1 min read # 软件使用

  周末简单看了下Tkinter库的帮助文档，写了一个小程序，记录鼠标点击按钮的次数，代码如下：

import tkinter as tk

top = tk.Tk()
n = 0
def ph():
    global n
    n = n + 1
    lab.configure(text =str(n) )
    return 0
butn =tk.Button(top,text = "click",command=ph)
lab = tk.Label(top,text = str(n))
lab.pack()
butn.pack()
top.mainloop()
